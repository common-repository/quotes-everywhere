== Description ==

Every Wordpress user knows the 'Hello Dolly' plugin. Ever wondered how you got your own favorite song lyrics, or inspirational quotes anywhere on your site?
Great! Welcome to 'Quotes Everywhere!', it's like 'Hello Dolly', but with your own quotes, everywhere where you want, and way more customisable!
This plugin could not have been created without 'Hello Dolly' plugin.
[Created by Sam Simons.](http://sam-simons.nl/) - [More about this plugin] (http://sam-simons.nl/webdev/quotes-everywhere/)

== Installation ==

1. Install the plugin (you know how)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('quote_here'); ?>` everywhere you want to appear the quotes
4. ATTENTION -> Fix the settings (the tags and the style). NB: The quotes will NOT APPEAR when the tags are not set!
